# AIBIR – AI-Based Internship Recommendation Engine

AIBIR is an AI-powered system that recommends relevant internships to students by semantically matching resumes with internship requirements.

Problem Statement:
SIH 2025 – AI-Based Internship Recommendation Engine (Ministry of Corporate Affairs)

Tech Stack:
- Backend: Python, FastAPI
- AI/ML: NLP, Semantic Matching
- Frontend: React
